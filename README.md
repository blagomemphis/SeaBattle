# Морской бой
Морской бой - Морской бой - игра для двух игроков на сетке. 
Цель: обнаружить и уничтожить корабли противника. 
Игроки атакуют координаты на поле. 
Корабли не могут пересекаться. 
Побеждает тот, кто первым уничтожит все корабли противника.

### Установка
- Создайте проект в Visual Studio.
- Чтобы запустить игру, запустите проект. 
- После запуска игры, пользователь увидит свое и вражеское поле и надпись о вводе координат для расстановки корабля и определения его типа. 
- Когда корабли всех размеров будут расставлены, пользователю будет предложено ввести координаты вражеского поля. 
- Пользователь и компьютер по очереди совершают выстрелы до тех пор, пока все корабли соперника не будут потоплены.

### Детали игры
В начале игры каждый из игроков размещает свои корабли на своем поле. Корабли размещаются вертикально или горизонтально. 
После размещения кораблей начинается основная часть игры. Игроки по очереди делают выстрелы, указывая координаты на поле противника. 
Если выстрел попадает в корабль противника, то на поле отмечается попадание символом "X". 
Если выстрел промахивается, то на поле отмечается промах символом "-". Когда все клетки корабля подбиты, он считается потопленным и отмечается символом "*". 
Игра продолжается до тех пор, пока у одного из игроков не будут потоплены все корабли.

### Бой
Чтобы расставить корабли, пользователь должен вводить координаты начальной точки (например, 0,0) и определить тип корабля (0 - вертикальный, 1 - горизонтальный). 
Затем пользователь также вводит координаты выстрела (например, 0,0). После каждого ввода выстрела, пользователь полуает информацию о своем выстреле: совершено попадание по кораблю противника или промах. 
И о выстреле противника аналогично. Затем пользователь нажимает Enter для очистки полей и совершения следующего выстрела. 
По завершению игры, пользователь увидит надпись о победе, в случае потопления всех кораблей соперника или о поражении, если соперник опередил его и потопил корабли первее. 
Также будет выведена надпись о нажатии на любую кнопку для завершения игры.
